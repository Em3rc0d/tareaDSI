Lista de APIs y Servicios

  • Exchange Rate API: Proporciona tipos de cambio de PEN a otras monedas. https://v6.exchangerate-api.com
  • Ipify API: Obtiene la IP del usuario. https://api64.ipify.org
  • IPWhoIs API: Proporciona información geográfica de la IP del usuario. https://ipwho.is
  • Lyrics.ovh API: Obtiene letras de canciones según el artista y el título. https://api.lyrics.ovh
  • Nationalize.io API: Predice la nacionalidad de una persona según su nombre. https://api.nationalize.io
  • RestCountries API: Proporciona información sobre todos los países, incluyendo banderas. https://restcountries.com/v3.1/all
  • RandomUser API: Genera información aleatoria de usuario. https://randomuser.me/api/
  • TimeAPI: Proporciona la hora local para una zona específica. https://timeapi.io
  • Open Trivia Database API: Proporciona preguntas de trivia. https://opentdb.com
  • Vercel API: Servicio de Vercel para despliegue y administración. https://tarea-dsi.vercel.app
  • Auth0 API: Proporciona autenticación segura para usuarios. https://auth0.com
